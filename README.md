# MercariSizer

スマホのカメラだけで物体のサイズを測定し、メルカリ便の最適な配送サイズをおすすめするiOSアプリです。

## 概要

ARKit の平面検出を使い、定規なし・参照物なしでサイズを測定します。
免許証スキャンのように、カメラを動かす手順に沿って操作するだけで OK です。

## 測定フロー

1. **平面検出** — 物体を平らな面に置き、カメラをゆっくり動かす（緑のグリッドが出たら OK）
2. **角をタップ（1点目）** — 真上から見て物体の一角をタップ
3. **角をタップ（2点目）** — 対角をタップ → 長さ・幅が確定
4. **高さをタップ** — 横から見て物体の最上部をタップ → 高さが確定
5. **結果表示** — 寸法とおすすめメルカリ便サイズ（最安順）を表示

## 対応する配送サービス

| サービス | サイズ |
|---------|--------|
| らくらくメルカリ便 | ネコポス / 宅急便コンパクト / 宅急便 60〜160 |
| ゆうゆうメルカリ便 | ゆうパケット / ゆうパック 60〜100 |

## 要件

- iOS 16.0 以上
- カメラ付き iPhone（LiDAR 非必須、あれば精度向上）
- Xcode 15 以上

## ビルド方法

```bash
# xcodegen でプロジェクト生成（初回のみ）
xcodegen generate

# Xcode で開く
open MercariSizer.xcodeproj
```

Xcode で実機ターゲットを選択して Run（ARKit はシミュレータ非対応）。

## 注意事項

- 送料は目安です。メルカリアプリ上の実際の料金を必ず確認してください。
- 測定精度は ±2〜5cm 程度です。サイズ境界付近は余裕を持って判断してください。
